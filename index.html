Presentation Problems

Does the venue computer have compatible powerpoint version installed?

How to promptly share slides with all audiences using different OS?

Reveal.js is a presentation framework that utilizes HTML5, CSS and JavaScript to render portions of a webpage as individual slides.

Reveal.js lets you build your presentations as a website, so all you need to present is a browser.

Anything that runs on the web can run on the slides. Reveal.js can run on mobile devices without the need to install an app
