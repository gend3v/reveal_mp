What is Reveal.js?

A presentation framework that utilizes HTML5, CSS and JavaScript to render portions of a webpage as individual slides.

Will my slides file be recognized by the shared computer at the venue?

How to share slides with all audiences using different OS?

Will my slides be accessible to all individuals?

Most important one, how to make my slides interesting & engaging and not just another boring presentation?

Reveal.js lets you build your presentations as a website, so all you need to present is a browser.

Anything that runs on the web can run on the slides. Reveal.js is can run on mobile devices without the need to install an app

Goals

  • Extend/customize the framework to conform to ERNI branding
  • Use the framework to speak visually
  • Create engaging and interactive presentations
  • Be able to embed different media on presentation
  • Utilize ERNI’s web development capabilities in marketing

Scope

  • Marketing Presentations
  • Not meant to replace powerpoint EMOT
  • Runs on the browser
  • Written in HTML5, CSS3 and Javascript

Expected Result

  • This presentation
  • Simple GUI to show capabilities of the framework

Limitations

  • Old browsers does not support like IE8 below
  • No opensource GUI editor
  • Assets and Themes needs to be created
